# ⚡ Standalone Off-Grid Solar Inverter with Enhanced Transient Stability Using Supercapacitors and ESP32-Based Smart Control

> A Final Year Project focused on improving the transient stability, efficiency, and reliability of standalone solar power systems through intelligent control and energy storage integration.

---

## 📖 Project Overview

This project presents the design and development of a **Standalone Off-Grid Solar Inverter** capable of supplying stable AC power from a photovoltaic (PV) system without reliance on the utility grid.

To overcome voltage fluctuations and inrush current during inductive load startup, **supercapacitors** are integrated in parallel with the battery bank. An **ESP32 microcontroller** provides intelligent monitoring and smart cooling control, while a **Maximum Power Point Tracking (MPPT)** algorithm maximizes solar energy utilization.

The system was modeled and analyzed using **MATLAB/Simulink** to evaluate transient response, voltage regulation, and overall system performance.

---

# 🎯 Objectives

- Design an efficient standalone off-grid solar inverter.
- Improve transient stability during motor startup.
- Reduce voltage sag using supercapacitor integration.
- Maximize solar energy harvesting using MPPT.
- Develop an ESP32-based smart monitoring and cooling system.
- Simulate and validate system performance using MATLAB/Simulink.
